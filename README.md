# Aditya_Yolov11 🚀

## 📌 Overview
**Aditya_Yolov11** is a custom object detection project built on the powerful **YOLOv11** architecture.  
This project demonstrates how to train, test, and use an object detector for real-time detection tasks like identifying objects in images or video streams.

Object detection is widely used in autonomous systems, surveillance, robotics, and smart applications.

---

## 🧠 Key Features
✅ Train your own dataset with YOLOv11  
✅ Test object detection on images & webcam  
✅ Real-time output visualization  
✅ Easy to extend for custom classes

---

## 🛠 Tech Stack
| Technology | Purpose |
|------------|---------|
| Python | Core language |
| PyTorch | Deep learning framework |
| YOLOv11 | Latest YOLO object detection |
| OpenCV | Image & video processing |
| CUDA (optional) | GPU acceleration (if available) |

## 🗂 Repository Contents
📦 `data/` – dataset images & labels  
📦 `models/` – trained YOLOv11 models  
📦 `scripts/` – training/testing utilities  
📄 `train.py` – train model  
📄 `detect.py` – run detection
